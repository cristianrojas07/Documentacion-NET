Este directorio contiene la documentacion de **Microsoft.RulesEngine**, un motor de evaluacion de reglas de negocio basado en expresiones para .NET.

La documentacion esta disponible en dos versiones segun el nivel de profundidad que necesites:

## Versiones disponibles

| Version | Carpeta | Descripcion |
|---------|---------|-------------|
| **Extensive** | [`extensive/`](./extensive/) | Guia tecnica exhaustiva organizada por modulos. Cubre en profundidad la arquitectura interna, filosofia de diseno, cada funcionalidad del motor y casos de uso avanzados. Ideal para cuando necesitas entender el motor a fondo. |
| **Minimal** | [`minimal/`](./minimal/) | Guia concisa y estructurada con tabla de contenidos. Abarca los mismos temas centrales pero de forma mas directa y orientada a consulta rapida. Ideal para referencia del dia a dia. |

## Cual elegir

- Si estas evaluando adoptar la libreria o necesitas comprender su funcionamiento interno en detalle, empieza por la version **extensive**.
- Si ya conoces la libreria y necesitas recordar configuraciones, patrones o resolver dudas puntuales, la version **minimal** te sera mas util.
